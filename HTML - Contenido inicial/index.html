Contenidos iniciales a HTML.


Introducción:

Antes de comenzar, quiero que piensen en cualquier página web que visitan todos los días: una red social, una tienda online o incluso un diario digital.

¿Alguna vez se preguntaron cómo están construidas?

Detrás de cada página hay un lenguaje fundamental: HTML.

HTML (HyperText Markup Language) es el esqueleto de cualquier sitio web. No es un lenguaje de programación, sino un lenguaje de marcado que nos permite estructurar el contenido de una página. Es como los cimientos de una casa: define dónde estarán los textos, imágenes, botones y enlaces.

Para entender cómo funciona, vamos a empezar con su estructura básica. Todo documento HTML tiene una organización específica que le indica al navegador cómo debe mostrar el contenido. Veamos la estructura mínima de una página web: 

<!DOCTYPE html> <!-- Define que el documento es HTML -->
<html lang="es"> <!-- Etiqueta principal o raiz que encierra todo el contenido del documento-->
    <head>  <!-- Contiene información del documento, como el titulo, enlaces a CSS-->
    <meta charset="UTF-8"> <!-- Define la codificación de caracteres para soportar tildes y ñ-->
        <meta name="viewport" content="width-device-width, initial-scale=1.0"> <!-- Ajusta la visualización en dispositivos móviles--> 
        <title>Mi primera Documento HTML</title>   <!-- Define el título que aparece en la pestaña del navegador-->
    </head>
    <body> <!-- Contiene todo el contenido visible del documento-->
        
    </body>
</html>

Visual Studio Code:


Visual Studio Code

Para escribir y organizar nuestro código, vamos a usar Visual Studio Code (VS Code), un editor de texto avanzado que nos permite desarrollar con facilidad.

VS Code nos ofrece herramientas como resaltado de sintaxis, autocompletado y extensiones que facilitan el desarrollo de documentos.

Para comenzar, abriremos VS Code y crearemos una nueva carpeta para nuestro proyecto. Luego, dentro de esa carpeta, creamos un archivo llamado index.html.

Y escribimos la estructura básica de HTML que vimos anteriormente o, para hacerlo más rápido, utilizamos el atajo que se muestra a continuación.

Atajo para la estructura básica de HTML.

En el editor de código VS code, se puede generar automáticamente la estructura básica de un documento HTML utilizando un atajo de teclado. Para hacerlo, simplemente se escribe el signo de admiración ! y luego se presiona la tecla Tab (o Enter). Este atajo crea de forma automática el esqueleto inicial del documento HTML, incluyendo las etiquetas principales como <!DOCTYPE html>, <html>, <head>, y <body>, facilitando así el inicio del desarrollo de una página web.
